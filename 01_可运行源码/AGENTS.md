# NovaPilot 项目协作规则

## Agent skills

### Issue tracker

本项目使用 `.scratch/` 下的本地 Markdown 文件跟踪规格与实施任务。参见 `docs/agents/issue-tracker.md`。

### Triage labels

本项目使用五类默认 triage 标签：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。参见 `docs/agents/triage-labels.md`。

### Domain docs

本项目采用单上下文领域文档结构；开始产品或工程工作前读取根目录 `CONTEXT.md` 与相关 ADR。参见 `docs/agents/domain.md`。
