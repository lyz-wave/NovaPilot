# PRD 采用完整蓝图与可信 MVP 双层范围

NovaPilot PRD 同时描述未来 12 个月的完整产品蓝图和可在 12 周内验收的可信 MVP。这样既完整响应竞赛要求，又避免把四场景、三语种、生信执行和高阶自主进化全部塞入首期；首期聚焦 FFPE/RNA 的实验方案设计与平台选型，并打通循证回答、项目记忆、专家转接和授权式 CRM 转化。

## Consequences

完整蓝图中的能力不自动成为首期承诺；每项需求必须标注所属阶段，可信 MVP 的成功以安全与业务闭环指标验收，而不是以功能数量验收。

可信 MVP 明确不包含自动执行完整生信流程、自动控制湿实验设备、临床诊断或治疗建议、自动报价或签约、A3 自主多步科研规划、在线模型微调、生产工作流自改、实时语音或视频，以及企业级批次 SLA。
